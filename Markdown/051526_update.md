# Current Project State

JobIntel is evolving into a:

> Local-first autonomous semantic career intelligence system.

The architecture is intentionally:
- modular
- inspectable
- debuggable
- AI-maintainable
- incrementally expandable

The current focus is NOT autonomous job application.

The current focus is building the foundational semantic intelligence pipeline.

---

# Core Architectural Direction

The system is being designed around:

## 1. Semantic Identity
A structured candidate intelligence profile drives:
- job scoring
- semantic ranking
- future resume tailoring
- future cover letter generation
- filtering logic
- autonomous decision support

## 2. Local-First Execution
Primary stack:
- Python 3.12+
- FastAPI
- SQLite
- SQLAlchemy
- sentence-transformers
- Ollama
- python-docx

No cloud-native assumptions.
